Researchers

Researchers are invited to use IPM Decisions Platform and dashboards to improve existing pest DSS tools by adapting existing models from other regions/countries to suit the situation in their region/country by modifying the internal parameters of models underpinning the DSS tools without the need for programming skills (where these parameters have been made accessible). They will be able to rapidly test and evaluate these novel DSS combinations against current DSS tools using the data sources and data manipulation algorithms available through Platform.
