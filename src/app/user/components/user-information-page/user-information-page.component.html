Icons guide

The IPM Decisions platform provides different kinds of information to the user, through the use of clickable icons.
When an icon is clicked a popup window will appear, allowing the user to check infos, errors etc about the DSS

  • Informative icons :
    • Info : Provide a brief description about the DSS and other info.
  • Error status :
    • Error : An error occured while running the DSS, show the error message.
    • Warning : There is something to be check into the configurations of the DSS.
    • Information : Some information about the execution.
  • Syncronization icons :
    • Sync : The DSS is synchronizing with the server.
    • Scheduled : The DSS is scheduled for retrieve data, click to show the schedule time.
